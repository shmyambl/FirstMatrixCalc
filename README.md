
#  Matrix Calculator (Pure python & NumPy)

Этот проект представляет собой учебный матричный калькулятор с двумя реализациями: на чистом Python и на NumPy.

##  Возможности
1)   **Транспонирование**: Поворот матрицы (строки становятся столбцами).
2)   **Умножение**: Матричное умножение по правилам линейной алгебры.
3)   **Деление**: Реализовано через поиск обратной матрицы (m1 * m2⁻¹).
4)   **Защита от ошибок**: Встроенные проверки на корректность размеров и определитель матрицы (ValueError).

##  Структура проекта
1.   `matrix_scratch.py` — реализация алгоритмов на чистом Python (без сторонних библиотек).
2.   `matrix_numpy.py` — быстрая реализация с использованием библиотеки `numpy`.
3.   `test_matrix.py` — автоматические тесты для проверки корректности работы обеих версий.

##  Как запустить

### Требования
Для работы версии на NumPy и запуска тестов вам понадобится:
```bash
pip install numpy pytest
```

### Запуск тестов
Используется `pytest` для автоматической и подробной проверки кода:
```bash
pytest -v
```

### Использование в своем коде
```python
from matrix_scratch import Matrix # импортируем Класс калькулятора с методами для матриц

m1 = Matrix([[1, 2], [3, 4]]) # Создаем объект наследующий методы родительского класса
print(m1.transpose().data) # используем методы через .
```

##  Технические особенности

     Весь полезный код (классы, методы) пишите как обычно.
     Все примеры, черновики и принты для проверки работоспособности прячьте внутрь этого блока:
     class Matrix: 

         # полезная логика 

         pass

     if __name__ == "__main__": 

         # этот код увидете только вы при запуске файла 

         m = Matrix([[1, 2]]) 

         print("Проверка прошла успешно!") 

     Это делает ваш код профессиональным: его можно использовать как библиотеку, и он не будет «мусорить» при подключении к другим программам.
