
# 🧮 Matrix Calculator (Scratch & NumPy)

Этот проект представляет собой учебный матричный калькулятор с двумя реализациями: на чистом Python и с использованием библиотеки NumPy.

## ✨ Возможности
*   **Транспонирование**: Поворот матрицы (строки становятся столбцами).
*   **Умножение**: Матричное умножение по правилам линейной алгебры.
*   **Деление**: Реализовано через поиск обратной матрицы (m1 * m2⁻¹).
*   **Защита от ошибок**: Встроенные проверки на корректность размеров и определитель матрицы (ValueError).

## 📁 Структура проекта
*   `matrix_scratch.py` — реализация алгоритмов на чистом Python (без сторонних библиотек).
*   `matrix_numpy.py` — быстрая реализация с использованием библиотеки `numpy`.
*   `test_matrix.py` — автоматические тесты для проверки корректности работы обеих версий.

## 🚀 Как запустить

### Требования
Для работы версии на NumPy и запуска тестов вам понадобятся:
```bash
pip install numpy pytest
```

### Запуск тестов
Мы используем `pytest` для автоматической проверки кода:
```bash
pytest -v
```

### Использование в своем коде
```python
from matrix_scratch import Matrix

m1 = Matrix([[1, 2], [3, 4]])
print(m1.transpose().data)
```
